# Pammmuse_admin

개인 프로젝트로 진행

pammmuse 쇼핑몰의 admin 사이트 입니다.


# Description

- 개발 기간: 2023.03 ~ 2023.05

- 사용 기술

  - SpringBoot 2.7.9,  Apache Tomcat 9.0, BootStrap,  Mybatis, AWS S3
  - Java 11,  Ajax,  Jquery, JavaScript,  Git,  MVC Pattern
  - MySQL DataBase

- 구현 내용
  
  - admin 사이트
  
    - 관리자 로그인
  
    - 상품 등록, 삭제, 상세 조회
  
    - 공지 등록
  
    - 주문 현황, 주문 취소
